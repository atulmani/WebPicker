Rule Book

    Must Have in head section
  1. Charset metatag
    meta charset="utf-8"
  2. description metatag
    meta name="description" content="This is the description search using search engine so define accordingly"
  3. keywords metatag
    meta name="keywords" content="business domain specific keywords search by search engine"
  4. author metatag
    meta name="author" content="atul triatphi"
  5. viewport metatag
    meta name="viewport" content="width=device-width, initial-scale=1.0"
    Use below html tags instead of
  1. Italic text: use 'em' tags instead of 'i' tags
  2. Bold text: use 'strong' tags instead of 'b' tags
